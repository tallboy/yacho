## How This Notebook Works

Each domain is self-contained. Work has its own priority tracker and weeklies. Portuguese has its own weeklies. The side project tracker is lighter because it gets fewer hours per week. Travel is a single planning doc until it gets closer.

Cross-links connect the domains where they naturally overlap — Portuguese study feeds into the Lisbon trip, work stress shows up in weekly notes. The structure scales without becoming overhead because each folder only has what it needs.
